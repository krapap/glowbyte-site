## Хронология изменений

### 1. Структура и контент-план
Запрос: структура сайта IT-консалтинга, что должно быть на первом экране.
- Составлен план секций: хедер → hero → услуги → почему мы → кейсы → процесс → технологии → отзывы → CTA → футер.
- Разобран первый экран (Hero): заголовок-выгода, подзаголовок, CTA, триггеры доверия, визуал.
- Выбран формат: **подробный текстовый план**, стиль — строгий корпоративный.

### 2. Первая вёрстка по гайдбуку
- Собрана главная страница на фирменных цветах.
- Hero как обложка гайдбука: белый фон, рыжий акцент, геометрия (кольца, точки, круги).
- Секции: услуги, «почему мы» (синий блок), кейсы с метриками, процесс, технологии, CTA, футер.

### 3. Оранжевый градиент + мега-меню «Направления»
Запрос: красивый оранжевый градиент, в верхней панели «Направления», которые высвечиваются; контент направлений — из презентации.
- Hero переведён на живой mesh-градиент в фирменных оттенках.
- Добавлено **мега-меню «Направления»** в шапке + сетка из 16 направлений, подсвечивающихся при наведении.
- Цифры и описания направлений взяты из презентации о компании.

### 4. Упрощение перенасыщенного экрана
- Запрос: главный экран перенасыщен → убраны соты, блобы, обводка, стрелка; градиент успокоен.
- Вариант «гигантская типографика» с бегущей строкой клиентов — затем тоже упрощён.

### 5. Подключены настоящие логотипы
- Пользователь добавил SVG (рыжий и белый) в `assets gb/`.
- Скопированы в `glowbyte-site/assets/`, расставлены по правилам: белый логотип поверх цветного фона → рыжий на белой шапке; белый — в подвале.

### 6. Светлый tech-вариант (промежуточный)
По наводящим вопросам выбрано: современный tech, белый фон, заголовок слева + визуал справа, умеренная анимация.
- Hero: белый фон, справа — анимированный BI-дашборд (KPI, столбчатый график, плавающие бейджи).
- Добавлены count-up статистики и reveal-анимации при скролле.

### 7. Финальное направление — синяя тема ✅
Запрос: синяя тема с оранжевыми акцентами + градиент; убрать дашборд; добавить анимацию про большие данные.
- Hero: глубокий **синий градиент** (`#141d3a → #1d2a52 → #32487D`) с оранжевыми бликами.
- Заголовок «BI и Big Data» — оранжево-жёлтым градиентным текстом.
- **Анимация больших данных:** живая сеть узлов (data-network) на `<canvas>` — точки движутся и соединяются линиями, ~18% узлов рыжие со свечением.
- Синий градиент подхвачен в секциях «Почему мы» и CTA (синий → оранжевый).
- Результат принят: «очень хорошо получилось».

### 8. Доработки финальной версии
- Добавлен отступ между блоком «Почему мы» и формой CTA.
- Добавлена 17-я практика — **«Заказная разработка»**.
- Заголовок Hero увеличен (`clamp(42–70px)`).
- Правило **висящих предлогов**: короткие предлоги/союзы (в, и, до, для…) не оставляем в конце строки — приклеиваем к следующему слову через `&nbsp;`. Применено к заголовку и подзаголовку.
- В анимационном кружке вместо «1500+ инженеров» — **терминал AI-агента**, печатающий код (typewriter с подсветкой синтаксиса, мигающий курсор, вокруг — вращающиеся кольца).

## Итоговая структура страницы

1. **Хедер** — логотип, мега-меню «Направления» (17 практик с описаниями), О компании, Клиенты, Контакты, кнопка «Обсудить проект». Прозрачный поверх Hero → белый при скролле.
2. **Hero** — синий градиент + анимированная сеть данных; слева заголовок/подзаголовок/кнопки/«нам доверяют», справа AI-терминал с печатью кода.
3. **Статистика** — 2000+ · 17 · 400+ · 20 с анимацией count-up.
4. **Направления** — 17 карточек, подсвечиваются при наведении.
5. **Клиенты** — лидеры рынка (текстом).
6. **Почему мы** — синий градиентный блок, 4 показателя.
7. **CTA** — градиент синий → оранжевый, форма заявки (`mailto`).
8. **Футер** — логотип, направления, компания, контакты.

**Технологии анимации:** canvas-сеть узлов, typewriter-код AI-агента, count-up цифр, reveal при скролле, hover-эффекты карточек. Уважается `prefers-reduced-motion`. Адаптив для мобильных.
